# My-portfolio-

This repository contains a minimal, modern single-page portfolio scaffold.

Files added:

- [index.html](index.html) — main site markup
- [css/styles.css](css/styles.css) — styling and responsive layout
- [js/script.js](js/script.js) — small interactivity (menu, theme, smooth scroll)

Quick start:

1. Open [index.html](index.html) in your browser.
2. Edit the site content (your name, project links, email) directly in [index.html](index.html).

Customizing:

- Replace "Your Name" in [index.html](index.html) with your real name.
- Swap project placeholders or link to live demos in the Projects section.
- Update contact email in the Contact section.

If you want, I can add more sections, deploy scripts, or convert this into a React/Vite app.
